# 🎮 Kill Head

**Kill Head** es un videojuego educativo desarrollado para el curso de **Teoría de la Computación** de la Universidad La Salle.

El objetivo del juego es enseñar de forma dinámica e interactiva conceptos fundamentales como:

- Autómatas Finitos Deterministas (AFD)
- Autómatas Finitos No Deterministas (AFND)
- Expresiones Regulares
- Gramáticas Libres de Contexto (GLC)
- Autómatas de Pila (PDA)
- Máquinas de Turing (MT)

Los jugadores compiten localmente (1 vs 1) respondiendo preguntas para avanzar y conservar sus vidas.

# 🧠 Mecánica

- Ambos jugadores comparten la misma pantalla.
- Aparece una pregunta de Teoría de la Computación.
- Cada casilla representa una respuesta.
- El primer jugador que llegue a la respuesta correcta mantiene sus vidas.
- Elegir una respuesta incorrecta hace perder una vida.
- Gana el jugador que sobreviva hasta el final.

# 🏗 Arquitectura

El proyecto sigue el patrón **MVC (Modelo - Vista - Controlador)**.

- **Model:** Maneja los datos y la lógica.
- **View:** Dibuja la interfaz.
- **Controller:** Coordina toda la ejecución del juego.
